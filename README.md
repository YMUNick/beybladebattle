# BEYBLADE X 對戰計分板

一個仿官方平板風格的 **戰鬥陀螺（BEYBLADE X）對戰計分板** 網頁，適合實體對戰時用手機或平板即時計分。單一 `index.html`，開箱即用，無需安裝。

🔗 **線上遊玩：** https://ymunick.github.io/beybladebattle/

---

## 玩法說明

### 1. 賽前設定
- 分別輸入 **Player 1 / Player 2** 的選手名稱。
- 各自登錄 **出戰陀螺 × 3**：左欄選「戰刃」，右欄填「配置」（例：`3-60LR`）。
  - 戰刃資料同步自 [Beyblade X Tier 天梯站](https://stan-yao.github.io/beyblade_x_tier)。
- 選擇賽制：
  - **4 分制（標準）** — 先取 4 分者獲勝
  - **7 分制** — 先取 7 分者獲勝
- 按 **「進入對戰」** 開始。

### 2. 對戰計分
每一回合結束後，點對應選手一側的決勝按鈕加分：

| 決勝方式 | 英文 | 說明 | 得分 |
|---------|------|------|:---:|
| 迴轉勝 | **SPIN** | 對手先停止旋轉 | **+1** |
| 場外勝 | **OVER** | 把對手打出場外 | **+2** |
| 爆烈勝 | **BURST** | 使對手爆裂解體 | **+2** |
| 極限勝 | **XTREME** | 於 Xtreme 線終結 | **+3** |

- **平手**：雙方同時停止／同時出場，按中央「平手」。
- **發射失敗**：每位選手有兩格失敗紀錄可標記。
- **先取滿目標分數（4 或 7）者贏得整場對戰。**

### 3. 對戰輔助
- **回合倒數**：每回合開始有 `3 → 2 → 1 → GO SHOOT` 全螢幕倒數與方向指示。
- **切換出戰陀螺**：對戰中可隨時從底部牌組切換當前上場的陀螺。
- **復原**：誤觸時可還原上一步計分。
- **離開**：返回設定畫面。

### 4. 對戰紀錄
- 每場對戰結果會自動存進本機資料庫。
- 可在 **「對戰紀錄」** 頁檢視歷史戰績（陣容、每回合決勝方式與比分）。
- 支援 **匯出 / 匯入 TXT 資料庫**，方便備份或跨裝置轉移。

## 技術說明
- 純前端單檔（HTML + CSS + JavaScript），無後端、無建置流程。
- 陀螺資料於載入時從天梯資料庫即時抓取。
- 部署於 GitHub Pages。

## 本機執行
直接用瀏覽器開啟 `index.html` 即可，或啟動任意靜態伺服器：

```bash
python -m http.server 8000
# 開啟 http://localhost:8000
```
